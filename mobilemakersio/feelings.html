Intro to Android: Feelings

Goal

We are going to create an app that takes your name as an input, and uses a spinner (which is kind of like a dropdown menu) to print out your mood and an image associated with your mood!

Create the Project

  • Open Android Studio
  • Create a new project
  • Application name: Feelings
  • Company name: your name
  • Change your project location to where you want it to be located
  • NEXT






  • Make sure Phone and Tablet is checked
  • Minimum SDK: API 16: Android 4.1 (Jelly Bean)






  • Click on Empty Activity. Blank Activity actually has extra stuff in it that we don't need.
  • NEXT
  • Leave MainActivity as it is. MainActivity.java is the main java file we will be working with.
  • Make sure Generate Layout File is checked
  • FINISH



Buttons, TextViews, and EditViews







  • Open the activity_main.xml file
  • The Design tab is the main design editor
  • The Text tab is the XML, the code behind the design
  • There should already be a textview in the app
  • Move it to the center horizontally by adding android:layout_centerHorizontal = “true”
  • Change the font size android:textSize = “24sp” (sp stands for scale-independent and is good for scaling to other phone sizes)
  • Change the text android:text = “How are you feeling?”
  • If you go into the Design tab, you will be able to see the new text centered






  • Go to the Design tab
  • Drag and drop a button and another Plain TextView under the button
  • They each get default text. We can change the text in the Strings.xml file.
  • Navigate to the strings.xml file and ddd the following line to change the button text to ‘Mood’
  • Mood
  • Then add the following line to create an empty text that will be filled with text after the button is pressed.
  • Now go back to activity_main.xml
  • In the Text tab, change the button text to
  • android:text=”@string/mood_button"
  • Then add the following line so that when the button is pressed, the function findMood is called
  • android:onClick=”findMood”
  • And the new textview text to
  • android:text=”@string/feeling"
  • Then change the same TextView’s ID to feelingText
  • android:id=”@+id/feelingText”
  • Now go to MainActivity.java. We are going to create a function that fills in the text when we click the button.
  • public void findMood(View view) {
    TextView feeling = (TextView) findViewById(R.id.feelingText); //finds the textview by ID
    feeling.setText("I'm in the mood to create Android apps!"); //sets the text
    }
  • Go to Activity_main.xml and add a spinner. We will store the values for the spinner in an array in Strings.xml
  • Happy Sad Curious Angry
  • Go back to activity_main.xml and in the spinner add the following line to access the entries
  • android:entries="@array/moods"
  • Run your app and make sure you can see the values of your spinner
  • Now go to MainActivity.java and update your findMood() function to include the spinner’s selected item in the text.
  • Spinner moodSpinner = (Spinner) findViewById(R.id.spinner);
  • String moodValue = String.valueOf(moodSpinner.getSelectedItem());
  • feeling.setText("I'm in a " + moodValue + " mood");
  • Run the app
  • EditText
  • Navigate to activity_main.xml
  • In the Design tab, add a horizontal linear layout under the spinner so items can be next to each other
  • Inside of the linear layout, add a TextView
  • Then under Text Fields, and a Person Name EditText right next to it
  • In strings.xml add Name
  • In activity_main.xml update the TextView and EditText with the following
  • android:layout_width=”wrap_content”
    android:layout_heigh=”wrap_content”
    android:text=”@string/name_textview” />
    android:layout_width=”wrap_content”
    android:layout_heigh=”wrap_content”
    android:id=”@+id/name_editText”
    android:layout_weight=”1” />
  • Now let’s update MainActivity.java so our message is personalized with our name.
  • Update findMood()
  • EditText name = (EditText) findViewById(R.id.name_editText);
  • String nameValue = name.getText().toString();
  • feeling.setText(nameValue + " is in a " + moodValue + " mood");
